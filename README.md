# Site-carta de aniversário

Presente visual em formato de carta interativa.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Como personalizar

- O texto da carta fica em `index.html`, dentro dos blocos `<article class="letter-card">`.
- As cores e o visual ficam em `styles.css`.
- A navegação da carta fica em `script.js`.

Se quiser colocar o nome dela, troque frases como `meu xuxuzinho` ou `dona encrenca` no `index.html`.
